    iron-jade

    A flatiron jade web app template

    This project is a web application template for @flatiron and jade. It uses the jade.plugin, which creates a simple api to render jade views:

    app.render(this.res, 'index', { title: 'Hello World' });

    Technologies

    This is a template that can be used to create nodejs applications using

    • Node v0.6.17
    • Express v3.0.0rc2
    • Jade v0.26.0

    Requirements

    • NodeJs
    • Flatiron
    • Jade

    These will install with npm, just do 

    npm install

    In your project directory.

    Install, Build, Run, Test, and Watch

    ## Install nodejs and npm

git clone http://github.com/twilson63/iron-jade.git [project-name]
cd [project-name]
npm install

    Run

    node server.js

    Setup to deploy to nodejitsu

    npm install jitsu -g
jitsu deploy
